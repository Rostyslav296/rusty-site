# Rusty

Static site generated and maintained by `website_sdk` / `website_post`.

**Live URL (GitHub Pages):** https://Rostyslav296.github.io/rusty-site/

## Structure
```
src/
  index.html              Home page
  posts.html              Posts archive (matches sample-website)
  posts/*.html            Individual posts (managed by website_post)
  social.html             Socials page (optional)
  about.html              About page (optional)
  contact.html            Contact + form (optional)
  css/styles.css          All site styling (CSS variables on :root)
  js/main.js              Nav toggle, contact form, smooth scroll, lightbox
  images/                 Site imagery
  sitemap.xml             Autogenerated
  robots.txt              Autogenerated
  feed.xml                Autogenerated RSS
  README.md               This file
```

## Updating the site
Every update goes through one of these SDK calls, never by hand-editing
structural HTML:
```
website_sdk  update_metadata | update_page | add_page | customize_theme | rebuild_nav | save_file
website_post research_post | plan_post | draft_post | review_post | publish_post
```
Then:
```
website_sdk build         # regenerates sitemap + robots + feed, runs nav check
website_sdk review        # SEO + quality gate
website_sdk git_push      # commit + push to origin/main
```

## Custom domain
Set a custom domain by writing CNAME in src/ (website_sdk customize_theme custom_domain=your.domain),
and at your registrar point A records 185.199.108.153 / 109.153 / 110.153 / 111.153 at GitHub Pages,
with a CNAME record `www` → `Rostyslav296.github.io`.
